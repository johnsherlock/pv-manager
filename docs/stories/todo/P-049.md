# P-049: Build multi-investment recovery and payoff insight surfaces

## Objective

Implement the first multi-investment recovery/payoff surfaces so users can see
recovered value, repayment coverage, and a simple payoff estimate in the
signed-in app across more than one investment record.

## Scope

- Extend signed-in loaders/API payloads to expose the multi-investment insight
  data needed by the chosen screens
- Implement the first recovery/payback modules defined by `U-051`
- Surface:
  - recovered value from available recorded history
  - estimated total recovery since each investment date when needed
  - selected-period savings vs repayment coverage
  - simple payoff estimate based on historical savings behavior
- whole-system totals and per-investment detail where the chosen surface needs both
- Ensure missing-investment state degrades to clear setup prompts rather than fake zeroes
- Ensure incomplete-history state explains confidence limits honestly
- Reuse existing financial calculations where possible rather than creating a
  separate finance math path
- Add automated coverage for:
  - one-record installations
  - multi-record installations
  - upfront-only records
  - repayment-backed records
  - partial-history scenarios
  - selected-period repayment coverage
  - payoff-estimate labeling

## Out of Scope

- Advanced forecasting or weather-informed payoff prediction
- Full Data Health or outage reporting UI

## Acceptance Criteria

- Signed-in users with investment records can see the first recovery/payback insights
- Users without investment records see a prompt state instead of misleading values
- Partial-history users see recorded vs estimated recovery distinguished clearly
- Selected-period repayment coverage reflects the configured investment records accurately
- Whole-system totals and per-investment detail are both coherent where shown
- Payoff estimate is visibly labeled as approximate guidance
- Tests cover the main happy paths and partial-history cases

## Dependencies

- `P-051`
- `P-052`
- `U-051`
- Existing range/overview financial surfaces
