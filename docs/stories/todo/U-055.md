# U-055: Define immersion trade-off and surplus-allocation insights

## Objective

Define how Range History should explain the trade-off between exporting surplus
solar and diverting surplus into immersion so users can understand whether that
energy feels better used as hot water or export credit.

## Scope

- Define the first immersion-focused financial interpretation layer for Range
  History
- Define the core immersion trade-off metrics, including:
  - immersion diverted kWh
  - exported kWh
  - potential export credit foregone by immersion diversion
  - immersion vs export split of surplus energy
- Define how these metrics should be framed so the product treats immersion as
  a trade-off, not automatically as a mistake or warning
- Define whether the first pass should also estimate the value of hot-water
  offset, or explicitly defer that until a better heating-baseline model exists
- Define the best home for these insights within the broader Range History
  financial model, including:
  - which metrics belong in Energy value
  - which belong in a deeper supporting section rather than the top headline
- Define the chart/comparison direction clearly enough to implement, such as:
  - ratio/split view
  - period trend comparison
  - plain-English explanatory callout

## Out of Scope

- Appliance-level control recommendations
- Automated advice about changing immersion behavior
- Full water-heating economics modeling if the baseline is too weak to support
  a trustworthy first pass

## Acceptance Criteria

- The product has a documented first-pass immersion trade-off model for Range
  History
- The story defines which immersion metrics are worth surfacing and how they
  should be described in user language
- The first-pass model makes clear that immersion diversion has opportunity cost
  relative to export without implying it is always the worse choice
- The implementation direction is clear enough to build without reopening the
  conceptual model

## Dependencies

- `U-054`
- Existing daily summary fields for immersion and export
