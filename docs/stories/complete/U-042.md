# U-042: Energy charts — trend line and per-day stacked bars

## Objective

Add the two energy charts to the Range History page: the daily trend line chart
and the per-day stacked bar chart. These give users an at-a-glance view of how
import and generation tracked over the selected period.

## Scope

### Chart 1: Energy trend line
- Lines: Import kWh, Generation kWh, Export kWh (toggleable; import and
  generation on by default)
- X-axis: local date; Y-axis: kWh
- ECharts line chart with smooth rendering
- Data gaps (missing days) represented as breaks in the line, not interpolated
  — this matches v1 behaviour and makes the Eddi-offline gap visible without
  misleading the user about what was recorded
- Tooltip shows date + values for hovered point

### Chart 2: Per-day stacked bar
Two groups of bars per day (consistent with v1 month view):

**Group 1 — Import:**
- Single bar: total import kWh
- When P-039 band data is available in the series: stack as day / peak / night
  import with distinct colours
- When band data is absent: single uncoloured bar with a "simplified" label

**Group 2 — Generation breakdown:**
- Stack: Gen consumed + Exported + Immersion diverted
- This shows how generated energy was used each day

**Dropped from v1:** the third redundant "Overall Gen" bar is not included.

- Tooltip shows date + breakdown values on hover
- X-axis: local date; Y-axis: kWh
- Both groups share the same X-axis

### Shared behaviour
- Charts re-render when the period selector changes
- Both charts are ECharts canvas-rendered (consistent with Live and Historical Day)
- Series visibility is toggleable via a legend
- Charts are responsive — height reduces on mobile so the chart is visible above
  the fold

## Out of Scope

- Financial charts (U-043)
- Solar coverage and ratio charts (U-044)

## Acceptance Criteria

- Trend line chart renders import and generation lines for the selected period
- Data gaps show as line breaks, not interpolated connections
- Per-day bar chart shows import and generation breakdown per day
- Import bar stacks by rate band when band data is present in the API response
- The redundant third bar from v1 is absent
- Both charts re-render correctly when the period changes
- Tooltips work on desktop hover and mobile tap

## Dependencies

- `U-041` (page shell — charts mount inside the page)
- `P-039` (band data in series — needed for import rate-band stacking)
