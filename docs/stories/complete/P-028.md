# P-028: Add adjacent-day caching, prefetch, and swipe-ready navigation support for Historical Day

## Objective

Restore the snappy back/forward experience for recent day browsing by adding
adjacent-day caching and prefetch, and make that performance foundation power
touch swipe navigation on mobile and tablet so moving through nearby days feels
instant rather than server-bound.

## Scope

- Add an in-memory cache keyed by local date for single-day payloads
- Cache fetched historical day payloads for revisit/navigation reuse
- Prefetch the previous day and next eligible day around the selected day
- Use cached payloads for instant render when revisiting or stepping between
  nearby days
- Expose navigation behavior that can be used by both button navigation and
  swipe navigation

## Out of Scope

- Persistent or offline cache
- Multi-day/range cache
- Server-side storage changes
- New provider payload shapes

## Constraints

- Next-day prefetch must never go beyond today
- Live refresh behavior must remain separate from historical cache behavior
- Button and swipe navigation must share the same underlying navigation/cache
  path
- Prefetch should reduce repeat fetches rather than adding redundant network
  traffic

## Dependencies

- `P-023`
- `U-034`

## Acceptance Criteria

- Revisiting cached days avoids waiting on a fresh fetch
- Adjacent navigation feels immediate after prefetch in common cases
- No redundant requests are made for already cached days
- Swipe-triggered navigation uses the same cached/prefetched path as button
  navigation
- Common back/forward swiping on touch devices feels fast enough to browse
  several recent days in sequence

## Tests

- Cache-hit path for revisiting a day
- Adjacent prefetch trigger behavior
- Guard that future days are never prefetched
- Swipe followed by additional swipe uses prefetched data where available
- Button and swipe navigation share the same cached behavior

## Notes

- Use the existing single-day payload as the cache unit
- This story is not just a performance optimization; it is the interaction
  backbone for quick historical browsing on touch devices
